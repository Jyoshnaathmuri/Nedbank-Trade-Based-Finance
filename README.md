This project involves automating multiple Trade-Based Finance workflows for Nedbank, one of the largest financial institutions in Africa. The automations span Windows, web, and mainframe applications.

## Role  
**Role:** RPA Developer  
**Technologies:** UiPath, SAP, MS Excel, Email Automation, JavaScript  

## Responsibilities  
- Developed UiPath workflows for SAP, TXT, CSV, PDF, and email automation.  
- Configured and managed Work Queues and BOT Schedulers.  
- Deployed bots across Dev, UAT, and Production environments.  
- Created reusable components (login, reset, error handling).  
- Monitored and managed bots using Control Tower.  
- Used JavaScript for data extraction from web pages.  
- Implemented exception handling using Try/Catch and structured logging.
